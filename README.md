## pillpal

An app for managing a medicinal regiment.

### Running the React app

- **Install dependencies** (from the `pillpal` folder):

```bash
npm install
```

- **Start the dev server**:

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`) in your browser. You should see the **PillPal** welcome screen.
To simulate a reminder to take a medicine, go to http://localhost:5173/alert.

### Tech stack

- **Build tool**: Vite
- **Language**: TypeScript
- **UI library**: React 18
